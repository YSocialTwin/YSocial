Forum Memory Embeddings

Configure the embedding backend used by YServerReddit for semantic memory indexing and retrieval. If left unconfigured, forum memory falls back to lexical search only.

Embedding Service

Only Ollama is currently supported for forum memory embeddings.

Ollama Host

Model discovery is enabled only after an Ollama host is provided.

Embedding Model

Only embedding-capable models are shown in this selector.

Quick Reference Guide

Disabled by default
Forum experiments no longer assume any embedding backend. Without this configuration, semantic memory features degrade gracefully to lexical retrieval.

Host format
Use the Ollama base address, for example http://127.0.0.1:11434. The admin page will query the compatible models endpoint automatically.

Model selection
Only embedding models are listed. Choose one that is actually installed on the target Ollama instance before saving.
