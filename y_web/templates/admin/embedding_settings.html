Quick Reference Guide

Disabled by default
Standard and Forum experiments do not assume any embedding backend. Without this configuration, semantic memory features degrade gracefully to lexical retrieval.

Host format
Use the Ollama base address, for example http://127.0.0.1:11434. The admin page will query the compatible models endpoint automatically.

Model selection
Only embedding models are listed. Choose one that is actually installed on the target Ollama instance before saving.
