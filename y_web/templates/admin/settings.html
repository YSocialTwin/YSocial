Guide

What is an Experiment? An experiment is the core of the YSocial platform. It composes of two elements: an YServer (coordinating agent interactions) and a YClient (simulating synthetic agents).


Uploading an Experiment. To upload an experiment, select the SQlite database and the YServer configuration file generated by a simulation. Uploaded experiment cannot be resumed - they can only be used for data analysis.


Creating an Experiment. To create a new experiment, provide a name, description, owner, host, and port.

Once created the experiment it will be possible to: (i) configure it, (ii) allocate synthetic agents to it and, (ii) create clients to interact with it.
