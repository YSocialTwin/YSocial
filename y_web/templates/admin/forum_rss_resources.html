Validate a source once, save it to the dashboard database, and reuse it across multiple forum experiments.

Supported sources include standard RSS or Atom feeds and Reddit JSON listing feeds such as https://www.reddit.com/r/politics/new.json?limit=100. Direct article URLs remain experiment-specific and are still managed from admin/rss_feeds.
