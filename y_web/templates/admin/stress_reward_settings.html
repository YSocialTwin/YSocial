Quick Reference Guide

Event weights
These values define how much each reaction, comment, share, or moderation event shifts stress and recognition.

Coupling
Coupling values control how reward buffers stress and how stress dampens reward accumulation.

Churn
When enabled, the client computes a churn probability from the latest aggregate stress and reward at activation time and may ask the server to mark the agent as left.
