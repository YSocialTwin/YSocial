Quick Reference Guide

AdHoc Clients
Configure a client dedicated to plugin-backed ad hoc agent families. Each client targets one custom agent type and one population of the same type.

Agent Type and Population
Agent Type: Mandatory. Select one of the installed custom agent families.
Population: Mandatory. Only populations with the same custom type are available.

Simulation Parameters
Reuse the standard scheduling and runtime controls, without forum-specific memory, recsys, network, or archetype configuration.

Large Language Models
The LLM section appears only for agent types that declare LLM requirements in their plugin registry.
