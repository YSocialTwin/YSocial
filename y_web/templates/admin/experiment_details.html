Quick Reference Guide

Experiment Overview
Use the left column to inspect metadata, update topics, toggle annotations, and open advanced configuration such as prompts, embeddings, and stress/reward settings.

Update runtime annotations and memory in Experiment Configuration
Maintain the active simulation topics from the overview card
Open prompt and embedding settings from Additional Configuration
Use Management to download or remove the experiment

Server, Clients, and Analytics
The center column controls execution and the lower sections expose logs and diagnostics for both the server and active clients.

Start or stop the server from Server & Analysis Controls
Launch clients and open the web interface from the same control panel
